name: vibrant-ui-designer
description: Enforce high-end aesthetic standards for the UI of the hackathon project, focusing on modern, premium, and dynamic design principles.

When activated for frontend tasks:
1. Prioritize visually stunning design choices: vibrant harmonious color palettes, smooth gradients, and glassmorphism.
2. Incorporate subtle micro-animations and dynamic interactions to make the app feel alive and responsive.
3. Avoid generic placeholder colors, basic system fonts, or "minimum viable" aesthetics. Suggest and use modern web typography (e.g., Inter, Roboto).
4. Make design recommendations before or during implementation to ensure the interface wows the user and hackathon judges at first glance.
